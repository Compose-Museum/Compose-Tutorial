库​

项目名介绍
AccompanistJetpack Compose 的一些功能补充库
Ucrop可以完成图片的裁剪，对用户的图片上传功能有帮助(已知 Bilibili 在用)
Material Design安卓应用的设计标准，内涵丰富的图标按钮资源
CMaterialColors包含所有 MaterialDesignColor，在 Jetpack Compose 中轻松使用
Material Dialog含有颜色、时间选择对话框等等
CoilCompose 额外的图像库，支持显示 Svg Gif，支持显示网络资源的图片
Landscapist同样也是图像加载的库，支持使用 Glide, Coil, Fresco 等库
ComposeDataSaver在 Jetpack Compose 中优雅完成数据持久化
